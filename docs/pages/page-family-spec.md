# 页面族规格

## 首页 `/`
- 展示站点定位
- 提供热门动作标准和核心计算器入口
- 承担 Strength Level Calculator 的导流入口

## 计算器总览 `/calculators`
- 列出全部核心计算器
- 每个计算器提供用途说明与跳转入口

## FAQ `/faq`
- 只保留训练与计算有用的条目
- 删除账号、公司、联系支持等弱相关内容

## 聚合标准页
- 男/女切换
- 单位切换
- 列出热门动作标准入口

## 普通动作标准页
- Hero
- 男/女切换
- 全体用户摘要表
- 按体重 / 按年龄表
- 单项等级计算器
- FAQ/说明区

## 动作对比页
- 双动作名称与切换
- Swap 跳转
- 男性/女性对照表
- 单位切换
- 返回单项页入口

## 计算器页
- 统一站点壳
- 输入表单
- 结果卡片
- 相关页面互链
