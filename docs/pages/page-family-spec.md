# 页面族规格

## 首页 `/`
- 展示站点定位
- 提供热门动作标准和核心计算器入口
- 承担 Strength Level Calculator 的导流入口

## 计算器总览 `/calculators`
- 列出全部核心计算器
- 每个计算器提供用途说明与跳转入口

## FAQ `/faq`
- 只保留训练与计算有用的条目
- 删除账号、公司、联系支持等弱相关内容

## 内容覆盖清单 `/content-coverage`
- 展示普通动作页、动作对比页和静态页面的覆盖统计
- 明确列出当前保留的页面族
- 提供抽样入口，方便人工验收真实数据页

## 动作列表 `/exercise-library`
- 提供整站普通动作的搜索入口
- 支持中文、英文、slug 与缩写的模糊搜索
- 每个动作卡展示双语名称、动作说明和能力要求摘要
- 支持直接跳转到 kg / lb 标准页

## 聚合标准页
- 男/女切换
- 单位切换
- 列出热门动作标准入口

## 普通动作标准页
- Hero
- 动作名称按“中文（English）”展示，英文更小更弱化
- 男/女切换
- 全体用户摘要重量表
- 全体用户摘要体重倍数表
- 男 / 女按体重明细表
- 男 / 女按年龄明细表
- 单项等级计算器
- FAQ/说明区
- 相关动作对比互链

## 动作对比页
- 双动作名称与切换，采用“中文（English）”样式
- Swap 跳转
- 男性/女性对照表
- 单位切换
- 返回单项页入口
- 相关动作对比互链

## 计算器页
- 统一站点壳
- 输入表单
- 结果卡片
- 相关页面互链
